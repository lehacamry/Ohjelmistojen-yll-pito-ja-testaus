# Arithmetic Operations

## Description
This is a JavaScript library that provides basic arithmetic operations with error handling for non-number inputs and division by zero.

## Installation
- Clone the repository or extract the zip
- Run `npm install`

## How to run
- Main program: `npm start` or `node main.js`
- Unit tests: `npm test`

## Repository structure
- src/mylib.js – library code
- test/ – Mocha + Chai unit tests
- main.js – example usage
